# FlightDelay Pred

This project uses flight records and weather data to explore flight delay prediction in a notebook workflow.

## Contents

- `flight_delay_prediction.ipynb`
- `Train/`
- `Test/`
- `Weather/`

## Dataset Note

The old portfolio repo mixed raw data files and generated CSV outputs together. For this cleaned version, the generated CSV artifacts were removed and only the notebook plus source data folders were kept.

If you need the derived CSV files again, regenerate them from the notebook.
